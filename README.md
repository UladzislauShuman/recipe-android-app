# Приложение для хранения рецептов

## Введение

В один из дней я заметил на сайте Т-Образования новый курс по Android-разработке для начинающих.
И я решил -- почему бы и нет.

Прошел курс, но должной практики не получил, поэтому решил сделать простое приложение, которое будет 
мне как пример

## О приложении

Как следует из названия, приложение предназначено для хранения рецептов.

## Функционал

В рамках курса были рассмотрены такие темы как:
- Работа с локальной БД и Room
- DI на примере Jetpack Hilt
- Работа с сетью (Retrofit и OkHttp)
- Clean Architecture и архитектурные паттерны (все самые популярные, но далее я использовал MVVM)
- Хоть в курсе рассматривали XML для интерфейса, я решил воспользоваться Jetpack Compose
- Навигация
- Ну и конечно основы Kotlin 

И я хотел применить как можно больше из этого списка, и вроде как получилось 

#### Что можно делать в приложении
- Искать рецепты в интернете (воспользовался открытым API [TheMealDB](https://www.themealdb.com/api.php))
- Добавлять их в "Избранное"
- Добавлять свои рецепты
- Чтобы не потерять свой список рецептов/ избранные, добавил функцию Бекапа (zip-файл сохраняется в Google Drive 
по пути, который укажет пользователь)
